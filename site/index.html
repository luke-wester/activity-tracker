Setup

  1. Download the latest Mac app from GitHub Releases.
  2. Open the app and allow macOS permissions when prompted.
  3. Click Start Tracking to begin logging activity.
  4. Use Open Live Report to review charts and recommendations.

What Users Need To Know

  • This app is for macOS.
  • It cannot run purely in the browser because browsers cannot observe system-wide app activity.
  • Users need to download the desktop app to use the tracker.
  • The web page is the download and onboarding layer.
